# WebGL Block Breaker Game 

This project is a Block Breaker (Brick Breaker) game built using **JavaScript and WebGL**.

## Description

The game features:

* Paddle and ball physics
* Brick collision system
* Score and lives tracking
* Start and pause functionality
* Modular architecture using JavaScript ES6 modules

## Academic Context

This project was developed during my **second year** of my **Bachelor of Engineering Technology in Computer Engineering**.

## Technologies Used

* JavaScript (ES6 Modules)
* WebGL (Graphics Rendering)
* HTML5 & CSS3

## How to Run

1. Open the project in VS Code
2. Install the *Live Server* extension
3. Right-click `index.html` → **Open with Live Server**

## Controls

* Arrow Left → Move paddle left
* Arrow Right → Move paddle right
* Start button → Begin game
* Pause button → Pause/Resume game
